# 2026年2月广西行 - 小红书风格UI设计规范

## 📋 项目概述

**项目名称**：2026年2月广西行  
**设计风格**：小红书风格  
**旅行时长**：5天（3天北海 + 2天防城港）  
**目标用户**：年轻旅行者（2位年轻人）  
**设计关键词**：年轻、时尚、活力、视觉冲击力

## 📝 2. 字体规范

### 字体家族
```css
/* 主字体 - 中文 */
font-family-chinese: -apple-system, BlinkMacSystemFont, "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", sans-serif

/* 英文和数字 */
font-family-english: -apple-system, BlinkMacSystemFont, "SF Pro Display", "Helvetica Neue", Arial, sans-serif

/* 衬线字体 - 标题装饰 */
font-family-serif: "Georgia", "Times New Roman", serif
```

### 字体层级
| 层级 | 字体大小 | 字重 | 行高 | 用途 |
|------|---------|------|------|------|
| **H1** | 48px | 700 | 1.2 | 页面主标题（Hero区） |
| **H2** | 32px | 700 | 1.3 | 章节标题 |
| **H3** | 24px | 600 | 1.4 | 小节标题 |
| **H4** | 20px | 600 | 1.5 | 卡片标题 |
| **Body Large** | 16px | 400 | 1.6 | 重要正文 |
| **Body** | 14px | 400 | 1.6 | 标准正文 |
| **Body Small** | 12px | 400 | 1.5 | 辅助文字、说明 |
| **Caption** | 10px | 400 | 1.4 | 标签、备注 |
| **Number Large** | 36px | 700 | 1.2 | 天数、价格 |
| **Number Medium** | 24px | 600 | 1.3 | 时长、评分 |

### 字体使用规则
- **标题**：使用PingFang SC，字重600-700，营造现代感
- **正文**：使用PingFang SC，字重400，确保易读性
- **数字**：使用SF Pro Display，字重600-700，增强视觉效果
- **英文标签**：使用Helvetica Neue，字重500，国际化风格
