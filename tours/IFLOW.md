# 旅程规划与记录项目

## 项目概述
这是一个旅程规划和旅程记录的工作项目，用于完成旅程规划工作并生成可直接部署到 Web 的旅游计划。

## 项目结构
每个旅程对应一个独立的文件目录，例如：
- `2026年2月广西行/`
- 其他旅行计划目录

每个旅程目录下包含该旅行计划的相关文件：
- HTML 文件
- CSS 样式文件
- JavaScript 交互文件
- 图片等资源文件

## 技术栈
- 静态网页技术（HTML/CSS/JavaScript）
- 可直接部署到 Web 服务器

## 项目目标
- 规划和记录旅行行程
- 生成美观、实用的旅行计划网页
- 便于分享和部署
